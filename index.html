Passo 1

Como funciona um Computador?

Aprendemos os passos seguros e corretos para ligar e desligar o computador usando o Windows, evitando puxar o cabo da tomada para proteger o sistema e evitar desligar ele pelo botão, mas sim pelo computador.

Atenção!
❌ Nunca desligue o computador puxando o cabo da tomada.
❌ Não pressione o botão de ligar para desligar diretamente (a menos que o computador esteja travado).
✔ Sempre use a opção Desligar no sistema.

Editor de Texto

Focamos em transformar uma folha em branco em um documento profissional, organizado e bem estruturado.

  • Formatação de Texto: Aprendemos a dominar o uso de Negrito (Ctrl + B), Itálico (Ctrl + I) e Sublinhado para dar destaque às ideias importantes.
  • Alinhamento Profissional: Como alinhar textos à esquerda, centralizar títulos, alinhar à direita e usar o formato Justificado para deixar trabalhos escolares e documentos impecáveis.
  • Estética e Fontes: Escolha adequada de fontes (como Arial e Times New Roman) e tamanhos adequados para títulos e corpo de texto.
  • Inserção de Elementos: Como adicionar imagens, criar tabelas organizadas e formatar listas com marcadores ou numeração.

    • Planilha

    Desmistificamos o bicho-papão das tabelas! Aprendemos que planilhas não são apenas para guardar dados, mas para fazer o computador trabalhar por nós.

  • Entendimento de Grade: O que são Linhas, Colunas e Células (e como localizá-las, como a famosa célula A1).
  • Fórmulas Matemáticas: Aprendemos a usar o sinal de igual (=) para iniciar uma fórmula e fazer contas automáticas de soma (=SOMA), subtração, multiplicação e divisão.
  • Formatação de Células: Como pintar tabelas, mesclar células para criar títulos bonitos e formatar números como Moeda (R$) para finanças pessoais.
  • Criação de Gráficos: Como transformar uma tabela cheia de números em um gráfico visual (de colunas ou pizza) fácil de entender.
